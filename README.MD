# 🏡 Ames Housing Data Analysis

مشروع تحليل استكشافي (EDA) لمجموعة بيانات **Ames Housing**، يهدف إلى فهم العوامل المؤثرة على أسعار العقارات.

## 🛠️ التقنيات المستخدمة
* **Python**
* **Pandas** (لتحليل البيانات والجداول)
* **Matplotlib & Seaborn** (للتصور البياني)

## 📌 الأهداف
* تنظيف ومعالجة البيانات العقارية.
* تحليل الارتباط بين المساحة، الموقع، وجودة البناء والسعر النهائي.
* استخراج رؤى إحصائية تدعم قرارات التقييم العقاري.
